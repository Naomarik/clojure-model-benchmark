## [0.1.0] - 2026-07-21

### Added

- Open 33-task one-shot Clojure correctness suite with deterministic graders.
- Ten stateful agentic debugging fixtures with public graders and references.
- OpenCode runners for fresh-session one-shot and temporary-workspace REPL runs.
- Separate correctness and heuristic REPL-use reporting.
- Fixture/reference validators and a no-model REPL lifecycle smoke mode.
- Matrix generators, sanitized exploratory development results, release documentation, and CI metadata.
